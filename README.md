# Badminton ELO Rating System

A web application for managing badminton players, generating matches, and tracking ELO ratings.

## Features

- **Player Management**: Add players with initial skill ratings
- **Event-Based Match Generation**: 
  - Select which players are participating in an event
  - Generate singles or doubles matches for selected players
  - Minimizes repeated partners for doubles matches
  - Minimizes repeated opponents for singles matches
- **Manual Match Entry**: Enter matches that weren't generated by the algorithm
- **Score Entry**: Enter match scores and automatically update player ratings
- **Match Management**: Delete individual unplayed matches or delete all unplayed matches at once
- **ELO Rating System**: Automatic rating updates based on match results
- **Match History**: View completed matches with rating changes
- **Data Export/Import**: Export player data and match history to JSON files for manual editing
- **Data Persistence**: All data is saved locally in your browser

## How to Use

1. **Add Players**: 
   - Go to the "Players" tab
   - Enter a player name and initial rating (default: 1500)
   - Click "Add Player"

2. **Export/Import Data**:
   - Click "Export Data (JSON)" to download player data and match history
   - Edit the JSON file manually to change player ratings if needed
   - Click "Import Data (JSON)" to load data from a JSON file

3. **Generate Matches**:
   - Go to the "Generate Matches" tab
   - Select which players are participating in the event (checkboxes)
   - Use "Select All" or "Deselect All" to quickly manage player selection
   - Select match type (Singles or Doubles)
   - Click "Generate Matches"
   - The system will create matches that minimize repeated pairings/opponents
   - New matches are added to existing unplayed matches

4. **Manage Matches**:
   - View all matches in the "Generate Matches" tab
   - Delete individual unplayed matches using the "Delete Match" button
   - Use "Delete All Unplayed Matches" to remove all pending matches at once
   - Completed matches cannot be deleted (they're in match history)

5. **Manual Match Entry**:
   - Go to the "Manual Match" tab
   - Select match type (Singles or Doubles)
   - Select players for each team
   - Enter scores
   - Click "Submit Match"
   - The match will be added to history and ratings will be updated

6. **Enter Scores**:
   - Go to the "Enter Scores" tab
   - Enter scores for pending matches
   - Click "Submit Score"
   - Player ratings will be automatically updated

7. **View History**:
   - Go to the "Match History" tab
   - View all completed matches and rating changes

## ELO Rating System

The application uses a standard ELO rating system with:
- **Initial Rating**: 1500 (configurable)
- **K-Factor**: 32 (standard for most competitive games)
- **Rating Calculation**: Based on expected vs actual match outcomes

## Data Management

- **Export**: Click "Export Data (JSON)" to download a JSON file containing all players and match history
- **Manual Editing**: Open the JSON file in a text editor to manually modify player ratings
- **Import**: Click "Import Data (JSON)" to load data from a JSON file (replaces current data)
- **Local Storage**: Data is automatically saved to browser localStorage as you use the app

## Technical Details

- Pure HTML, CSS, and JavaScript (no dependencies)
- Data stored in browser localStorage with export/import to JSON files
- Responsive design for mobile and desktop
- Modern, clean UI with gradient styling

## Getting Started

Simply open `index.html` in your web browser. No installation or server required!
